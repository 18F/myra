Start Saving for Retirement Now

Saving for retirement might seem complicated, but myRA makes it easy. You set your savings goal and decide how much you want to save—as little as a few dollars a month up to $5,500 per year.

myRA is a new way to kick start your savings by helping you set aside a few dollars per month. The more you can save the better. And the sooner you start saving, the more time there will be for your savings to grow, giving you more money in retirement.

For example, if you start putting away $100 a month now and do so for the next 30 years, you would earn $25,566 in interest for a total $73,566 in savings. But if you start putting away $100 a month now and only do so for the next 20 years, you would earn $13,345 in interest for a total $49,345 in savings. That's $12,221 in interest you would miss.
