myRA Resources for Employers

Helping your employees start saving for retirement with myRA is easy.

We’ve created tools you can use to help them learn about myRA and get signed up.

Choose the tools that work best for you

  1. Posters (green or blue) - A larger piece to hang in the workplace, intended to grab employees’ attention.
  2. Brochure - A double-sided brochure that can be printed or emailed.
  3. Infographic - Helps explain myRA in simple, straightforward language. Can be used online or printed.
  4. Web Banner - Directs people to myRA.treasury.gov. For use on a company intranet site.
  5. Top Questions About myRA - Frequently asked questions about myRA that can be printed or emailed.
  6. Intranet Content - If you have an internal company website (intranet) for employees, you can post this information about myRA.
  7. Employee Email on myRA - Use this to explain myRA to employees and encourage them to sign up.
  8. Employee Meeting Toolkit - Use these resources to set up a meeting with employees about myRA.
  9. Guide for Employers - A guide to help participating employers introduce myRA to their employees.
  10. PowerPoint Presentation - Use this brief PowerPoint presentation in meetings to explain the features and benefits of myRA and how to sign up.
  11. Key Fact Sheet About myRA - Use these facts along with the PowerPoint presentation or separately as a way to answer questions about myRA.

We appreciate your feedback. Please let us know if you have thoughts on how we can improve these resources by emailing myra@treasury.gov.
